## 中文可读性检查
- 是否中文优先：是。正文、按钮语义、卡片标题和风险提示均为中文优先。
- 是否仍有难懂英文：少量保留，主要是 Demo / Mock / Sample / Model Demo / ELO / FIFA / JSON 等数据或技术标识。
- 数据标签是否清楚：是。文本报告包含“数据类型”，分享卡包含“数据类型”badge，并保留原始英文数据层说明。
- 风险提示是否清楚：是。文本和图片均包含“不构成投注建议”或等价中文风险提示。

## 给中文用户看的测试问题
1. 你 30 秒内能看懂这张图/这段报告在讲什么吗？
2. 哪个最有用：今日情报、单场分析、球队报告？
3. 哪个最没必要？
4. 图片信息会不会太多？
5. 你会不会愿意每天看一版“世界杯今日情报”？
6. 你有没有误以为这是在保证预测结果？
7. “模拟数据 / 示例赔率 / 非实时”这些标签你看得懂吗？
8. 如果继续做，你最想增加什么：真实赛程、真实阵容伤病、真实赔率、中文短解释、短视频版，还是别的？
